
jupyter_clickable_image_widget
=====================================

Version: |release|

A clickable image widget for Jupyter.


Quickstart
----------

To get started with jupyter_clickable_image_widget, install with pip::

    pip install jupyter_clickable_image_widget

or with conda::

    conda install jupyter_clickable_image_widget
